# Shared

Общие компоненты. Содержит переиспользуемые элементы системы:
- contracts: Реестр входящих и исходящих контрактов данных
- data-storage: Общее агрегированное хранилище (read-only)
- data-models: Общие модели данных
- utils: Общие утилиты
- components: Переиспользуемые UI-компоненты

## Синхронизация с сервером

Для синхронизации репозитория с сервером используется автоматический процесс, который дублирует содержимое репозитория на удаленный сервер. Это обеспечивает резервное копирование и доступ к актуальной версии кода вне зависимости от состояния локального окружения.
