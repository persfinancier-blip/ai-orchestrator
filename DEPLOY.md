# Публикация страницы во внешний доступ

## 1) Локальный запуск страницы

```bash
python -m http.server 4173 -d site
```

После запуска страница доступна локально на:

- http://localhost:4173

## 2) Публикация наружу (туннель)

Нужен любой туннельный сервис. Например:

- `cloudflared tunnel --url http://localhost:4173`
- `ngrok http 4173`

Сервис вернёт публичный URL, по которому страницу можно открыть «со стороны».
