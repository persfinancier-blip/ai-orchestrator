# Tooling

Блок 3: Фабрика инструментов. Преобразует аналитические инсайты в готовые к исполнению инструменты и тактики.

## Strategy Tooling

Папка strategy-tooling содержит инструменты для работы с данными в рамках стратегического планирования.

### Data Platform

Платформа данных модуля включает в себя следующие компоненты:

- api: API компоненты модуля
  - queries: Получение данных из внешних API
  - commands: Отправка команд
  - command-log: Аудит отправленных команд
  
- connectors: Коннекторы для взаимодействия
  - inbound: Входящие соединения
    - contracts: Контракты входящих данных
    - events: События от других модулей
  - outbound: Исходящие соединения
    - contracts: Контракты исходящих данных
    - events: События для других модулей
