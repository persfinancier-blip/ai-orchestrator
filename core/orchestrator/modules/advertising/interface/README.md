# Интерфейс модуля Advertising

## Описание

Интерфейс модуля управления рекламой для AI Orchestrator. Реализован с использованием Svelte и Vite.

## Структура проекта

```
interface/
├── components/          # Компоненты интерфейса
│   ├── layout/         # Компоненты макета
│   ├── table/           # Компоненты таблицы
│   ├── campaign/        # Компоненты управления РК
│   ├── words/           # Компоненты управления словами
│   ├── actions/         # Компоненты массовых действий
│   └── status/          # Компоненты состояний
├── stores/              # Хранилища состояния
├── App.svelte           # Корневой компонент
├── main.js              # Точка входа
├── index.html           # HTML-шаблон
├── vite.config.js       # Конфигурация Vite
├── svelte.config.js     # Конфигурация Svelte
└── package.json         # Зависимости и скрипты
```

## Запуск интерфейса

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Интерфейс будет доступен по адресу: http://localhost:3000

### 3. Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в директории `dist/`.

### 4. Предварительный просмотр сборки

```bash
npm run preview
```

## Компоненты интерфейса

### Основные экраны
- `AdvertisingDashboard` - главный экран управления рекламой
- `ControlBar` - панель управления с фильтрами
- `PivotGroupPanel` - панель группировки данных

### Таблица артикулов
- `ArticlesTable` - таблица артикулов
- `ArticleRow` - строка артикула
- `GroupRowHeader` - заголовок группы

### Управление РК
- `CampaignToggleCell` - ячейка переключателей РК
- `ToggleTree` - древовидные переключатели
- `RadioToggleGroup` - радио-переключатели
- `PlacementToggleRow` - переключатели плейсментов

### Управление словами
- `WordsCellSummary` - сводка по словам
- `InspectorDrawer` - боковая панель управления
- `ClusterBidsTable` - таблица ставок
- `MinusPhrasesManager` - управление минус-фразами

### Массовые действия
- `BulkActionBar` - панель массовых действий
- `GroupContextMenu` - контекстное меню групп

### Состояния
- `SyncStatusBadge` - индикаторы статуса
- `ErrorTooltip` - подсказки ошибок
- `DraftChangesBanner` - баннер черновых изменений

## Хранилища

- `articlesStore.js` - данные артикулов
- `uiStateStore.js` - состояние интерфейса

## Стилистика

Интерфейс следует философии macOS:
- Минимализм и чистота
- Ясность и интуитивность
- Эстетика и плавные анимации
