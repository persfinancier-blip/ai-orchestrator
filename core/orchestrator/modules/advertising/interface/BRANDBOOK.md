# Брендбук интерфейса модуля Advertising

## Философия дизайна

Интерфейс модуля рекламы следует философии macOS - минимализм, ясность, эстетика. Все элементы интерфейса должны быть интуитивно понятными, с четкой визуальной иерархией и плавными анимациями.

## Основные принципы

### Минимализм
- Чистый дизайн с большим количеством свободного пространства
- Тонкие линии и сдержанные цвета
- Отсутствие лишних элементов

### Ясность
- Интуитивное взаимодействие с пользователем
- Предсказуемое поведение элементов
- Четкая визуальная иерархия

### Эстетика
- Плавные анимации
- Гармоничная цветовая палитра
- Сбалансированные композиции

## Цветовая палитра

### Основные цвета
- Белый (#FFFFFF) - основной фон
- Светло-серый (#F5F5F7) - вторичный фон
- Темно-серый (#86868B) - текст второстепенный
- Черный (#000000) - текст основной

### Акцентные цвета
- Синий (#0071E3) - основные действия
- Зеленый (#32D74B) - успех, включено
- Красный (#FF453A) - ошибка, удаление
- Желтый (#FFD60A) - предупреждение

## Типографика

### Шрифты
- Основной: SF Pro Display / SF Pro Text
- Альтернатива: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto

### Размеры текста
- Заголовки: 24px, 20px, 18px
- Основной текст: 14px
- Вспомогательный текст: 12px

## Компоненты интерфейса

### Кнопки
- Скругленные углы (8px)
- Высота: 36px
- Отступы: 16px по бокам
- Тень при наведении

### Поля ввода
- Высота: 36px
- Скругленные углы (6px)
- Тонкая граница (#D2D2D7)
- Анимация фокуса

### Карточки
- Скругленные углы (12px)
- Тень: 0 2px 10px rgba(0,0,0,0.05)
- Отступы: 20px

### Навигация
- Прозрачная по умолчанию
- Заливка при скролле
- Анимация перехода

## Иконки

### Стиль
- Линейные иконки
- Толщина линии: 2px
- Размеры: 16px, 20px, 24px
- Единая сетка 24x24px

## Анимации

### Продолжительность
- Вход: 0.2s
- Выход: 0.15s
- Перемещение: 0.3s

### Функция сглаживания
- ease-out для входа
- ease-in для выхода
- ease-in-out для перемещений

## Адаптивность

### Точки перехода
- Desktop: 1024px и выше
- Tablet: 768px - 1023px
- Mobile: до 767px

### Поведение
- Responsive сетка
- Адаптивные изображения
- Мобильные тап-таргеты (минимум 44px)
